# snake-water-gun-game
The Snake Water Gun Game is a simple Python command-line game where a player competes against the computer. The user selects Snake, Water, or Gun, and the computer randomly chooses one option. The program compares both choices using predefined rules to determine whether the player wins, loses, or the game ends in a draw.

# 🐍 Snake Water Gun Game (Python)

## 📌 Description
This is a simple command-line **Snake Water Gun** game built using Python. The game is similar to **Rock Paper Scissors**, where the player competes against the computer. The computer randomly selects Snake, Water, or Gun, and the program determines the winner based on predefined rules.

## 🎮 Game Rules
- Snake drinks Water → **Snake wins**
- Water drowns Gun → **Water wins**
- Gun kills Snake → **Gun wins**
- If both choose the same option → **Draw**

- Python 3
- Random module
